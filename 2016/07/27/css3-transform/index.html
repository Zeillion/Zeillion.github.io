css3的2d转换和3d转换

  • css
  • css3

前言

今天用到css3相关知识，所以把关于css3转换相前的内容做介绍，并总结。

兼容性

首先我们来看看各浏览器对css3转换的支持情况

css3-2d

css3-3d

结论： 如果不选配IE9以下的浏览器的话，我们还是可以放心使用css3的转换属性的。 我自己开发的页面是需要微信内置浏览器上运行，所以完全不需要去理会兼容性的问题。

写法

浏览器厂商以前就一直在实施CSS3，但它还未成为真正的标准。为此，当一些CSS3样式语法还存在波动时，它们提供针对流星器的前缀。现在主要流行的浏览器内核主要有：

Trident内核：主要代表为IE  -ms-
Gecko内核：主要代表为Firefox -moz-
Presto内核：主要代表为Opera -o-
Webkit内核：产要代表为Chrome和Safari  -webkit-

而这些不同内核的浏览器，CSS3属性（部分需要添加前缀的属性）对应需要添加不同的前缀，也将其称之为浏览器的私有前缀，添加上私有前缀之后的CSS3属性可以说是对应浏览器的私有属性。

所以我们css样式里面大致需要包含：

{
transform:rotate(30deg);
-ms-transform:rotate(30deg); /* IE 9 */
-moz-transform:rotate(30deg); /* Firefox */
-webkit-transform:rotate(30deg); /* Safari and Chrome */
-o-transform:rotate(30deg); /* Opera */
}

css3的写法需要先写transform然后指定转换名称和必要参数。

css3的2d转换

css3-2d转换有以下几种：

  • translate()
    通过 translate() 方法，元素从其当前位置移动，根据给定的 left（x 坐标） 和 top（y 坐标） 位置参数，这里位置参数也可以是面分比，这个百分比参照的是自身。我们平时经常使用些属性来进行元素居中。
    eg: 
    div
{
transform: translate(-50%,-50%);
}
  • rotate()
    通过 rotate() 方法，元素顺时针旋转给定的角度。允许负值，元素将逆时针旋转, 是根据元素的中心点旋转。
    eg: 
    div
{
transform: rotate(30deg);
}
  • scale()
  • skew()
  • matrix()
